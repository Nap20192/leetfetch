# leetfetch

## Краткое описание
CLI утилита для получения информации о профиле LeetCode.

## Возможности
- Получение информации о профиле
- Просмотр статистики задач
- Мониторинг прогресса

## Установка
```bash
go install github.com/Nap20192/leetfetch
```

## Использование
```bash
leetfetch --user [username]
```
